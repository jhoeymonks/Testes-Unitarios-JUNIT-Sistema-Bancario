Relatório de Teste de Qualidade - Funções TestPesquisaCliente e TestRemoveCliente do Sistema Bancario

Data: 24/05/2024

Autor: Jhoey Monks

Cenários de teste:

Objetivo Principal: Verificar se as funções isoladas de pesquisar um cliente e remover um cliente em um Sistema Bancario escrito em Java estão funcionando corretamente utilizando JUnit

1. testPesquisaCliente

Objetivo: Verificar se as função pesquisaCliente retorna o cliente correto com base no ID informado e se a idade do cliente está sendo validada corretamente durante a pesquisa.

  1. Criar dois clientes (cliente01 e cliente02) com IDs distintos.
  2. Adicionar os clientes à lista clientesDoBanco.
  3. Instanciar a classe GerenciadoraClientes com a lista clientesDoBanco.
  4. Executar a função pesquisaCliente com o ID 1.
  5. Verificar se o cliente retornado possui o ID 1 utilizando a asserção assertThat(cliente.getId(), is(1)).
  6. Verificar se o cliente retornado possui o email correto utilizando a asserção assertThat(cliente.getEmail(), is("jhoeymonks@gmail.com")).
  7. Verificar se o cliente retornado possui o nome correto utilizando a asserção assertThat(cliente.getNome(), is("Jhoey David")).
  8. Verificar se o cliente retornado possui a idade correta utilizando a asserção assertThat(cliente.getIdade(), is (20)).

Resultado: O teste foi parcialmente aprovado. A função pesquisaCliente retornou o cliente correto com o ID 1 e o email correto, mas a validação da idade falhou, pois a idade do cliente retornado (34) é diferente da idade esperada (20).

2. testRemoveCliente

Objetivo: Verificar se a função removeCliente remove o cliente correto da lista e se a lista é atualizada corretamente.

  1. Criar dois clientes (cliente01 e cliente02) com IDs distintos.
  2. Adicionar os clientes à lista clientesDoBanco.
  3. Instanciar a classe GerenciadoraClientes com a lista clientesDoBanco.
  4. Executar a função removeCliente com o ID 2.

Conclusão:

Os testes unitários para a função removeCliente da classe GerenciadoraClientes foram executados com sucesso. No entanto, a função pesquisaCliente, apesar de retornar o cliente correto, apresenta falha na validação da idade.
